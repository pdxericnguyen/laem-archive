# Using Codex with this repo

## Codex CLI (recommended)
1) Install Codex CLI
- npm: `npm i -g @openai/codex`
- Run: `codex`
(You'll sign in with your ChatGPT account or API key.)

Docs:
- Codex CLI quickstart: https://developers.openai.com/codex/quickstart/
- Codex CLI setup: https://developers.openai.com/codex/cli/
- CLI reference: https://developers.openai.com/codex/cli/reference/

## How to run Codex on this repo
1) Open terminal in the repo root.
2) Run: `codex`
3) Paste one task from `CODEX_TASKS.md`.

Tips:
- Ask Codex to run `npm run build` after changes.
- Prefer small, incremental tasks; review diffs.
